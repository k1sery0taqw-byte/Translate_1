# 🈯 Manhwa Auto Орчуулагч

Korean manhwa панелийг автоматаар Монгол хэл рүү орчуулдаг вэбсайт.

## ✨ Ямар зүйл хийдэг

1. Manhwa зураг upload хийнэ
2. Claude AI Korean текст бүгдийг уншина
3. Монгол хэл рүү орчуулна
4. Зурган дээрх bubble-д Монгол текст бичнэ
5. Орчуулсан зураг татаж авна

---

## 🚀 GitHub Pages дээр deploy хийх

### 1. Repository үүсгэх
```
GitHub → New Repository → manhwa-translator
```

### 2. index.html upload хийх
- `index.html` файлыг repository-д оруулна

### 3. GitHub Pages асаах
```
Settings → Pages → Source: Deploy from branch → main → / (root) → Save
```

### 4. Хаяг авах
```
https://YOUR-USERNAME.github.io/manhwa-translator
```

---

## 🔑 API Key авах

1. [console.anthropic.com](https://console.anthropic.com) дээр бүртгүүлнэ
2. **API Keys** → **Create Key**
3. `sk-ant-api03-...` гэсэн key-г хуулна
4. Сайт дээр нэг удаа оруулахад browser-д хадгалагдана

> ⚠️ API key нь нууц! GitHub-д commit хийхгүй, зөвхөн browser-д хадгалагдана.
